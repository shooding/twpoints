# 台灣紅利點數轉換一覽

![](pic.png)

計算台灣各種點數之間的最優交換比率(轉換路徑)。
計算是否有任何迴圈會造成無限增長點數？

# Install

```bash
brew install graphviz
```

# Usage

```bash
python points.py
```
